3. Servicios

3.1. Cuentas de Usuarios

En esta opción el Administrador del Sistema podrá Crear Cuentas de Usuarios y asignarlos a los diferentes tipos de Grupos de Usuarios donde se podrá seleccionar los Privilegios correspondientes.

REQUISITO: Para crear un Nuevo Usuario deberá ser Usuario Administrativo.

<CREACIÓN DE UNA CUENTA DE USUARIO>

  1. Hacer "Clic" en la opción [Crear Nuevo Usuario].
  2. Aparecerá una pantalla donde se proporcionará el Registro del Nuevo usuario. (Datos Personales)
  3. Escribir el Usuario o Login, es el nombre con el cual el nuevo usuario se identificará para realizar algún tramite.
  4. Escribir Contraseña, es la palabra clave que el nuevo usuario usará para acceder a su Cuenta Personal.
  5. "Seleccionar" el Grupo al que pertenecerá el nuevo usuario.
    • Administrtivo: Este grupo de usuarios, se le podrá asignar todo tipo de privilegios para el control del Sistema.
    • Profesores: Este Grupo de usuarios, no contiene la asignación de todos los privilegios, el administrador le asignará privilegios y restricciones por grupo de usuarios.
    • Alumnos de la Institución: Este Grupo de usuarios, no contiene la asignación de todos los privilegios, el administrador le asignará privilegios y restricciones por Grupo de usuarios.
  6. Escribir los siguientes Datos Personales. (Nombre, Género, Fecha de Nac., Direccion, Tel., Correo Electrónico)
  7. "Seleccionar" la Casilla Activo, esto indicará que el Nuevo Usuario tendrá su estado Activo.
  8. Si el Nuevo Usuario es Administrativo. Clic en la opción asignar privilegios ahora.
    • Se mostrará un lista completa de los Privilegios que se le pueden asignar a un Usuario Administrador. Seleccionar las casillas.
  9. Hacer "Clic" en la opción [Guardar Cambios].
  10. Mostrará un Mensaje de Alerta que indicará que los Datos Fueron Guardados y se enviará una notificación por email.

<CONSULTA DE BITÁCORA DE USUARIOS>

  1. El sistema mostrará una Lista completa de los usuarios registrados.
  2. Hacer "Clic" en la opción [Bit].
  3. El sistema mostrará la Bitácora de Actividades.
  4. "Seleccionar" las Fechas de Consulta, Desde y Hasta.
  5. "Seleccionar" el Nombre del Usuario.
  6. "Seleccionar" la Actividad.
  7. Hacer "Clic" en la opción [Consultar].
  8. El sistema mostrará los resultados que fueron encontrados.

<CAMBIO DE CONTRASEÑA>

  1. Hacer "Clic" en la opción [Passwd].
  2. Aparece un cuadro de diálogo.
  3. Escribir la nueva contraseña.
  4. Escribir la Confirmación de la Contraseña.
  5. Hacer "Clic" en la opción [Guardar cambios] y opción [Aceptar].
  6. Se mostrará un Mensaje de Alerta de que el cambio de contraseña se ha Realizado y será notificado por email al usuario.
