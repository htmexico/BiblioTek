2.3. Devoluciones Rápidas

En esta opción el Administrador del Sistema podrá realizar el proceso de Devolución de un Material en Préstamo de manera Rápida pidiendo el Item del Material a devolver.

REQUISITO: Para realizar una Devolución Rápida deberá contar con el Ítem del Material a Devolver.

  1. Escribir el Ítem del Material a Devolver, Teclee [Enter].
    • Si el Ítem no fue encontrado, el sistema mostrará un Mensaje de Alerta.
    • Si el Ítem fue encontrado, el sistema mostrará las características del Material. (Continuar Paso No. 2)
  2. Hacer "Clic" en la opción [Continuar].
  3. Aparecerá una pantalla mostrando los datos de Usuario y el Material en Préstamo.
  4. El Sistema seleccionará automáticamente el Material que será Devuelto, Fecha y Hora de Devolución.
  5. Hacer "Clic" en la opción [Devolver los Ítems Señalados].
  6. El Sistema mostrará una pantalla que indicará que la Devolución fué Registrada y si desea un Nuevo Proceso de Devolución.

 

NOTA: El Botón con el Símbolo de Interrogación (?) , puede ayudarnos a Localizar o Consultar un Usuario de la Institución o un Ítem de un Material.
