5. Configuración

5.7. Catalogo de Restricciones.

En esta opción el Administrador del Sistema podrá realizar la Configuración del Catalogo de Restricciones, alta de un nuevo tipo de Restricción, borrar y modificar.

REQUISITO: El administrador del sistema deberá tener los suficientes privilegios para tener acceso a esta opción.

<AGREGAR NUEVO TIPO DE SANCIÓN>

  1. Hacer "Cilc" en la opción [Nuevo Tipo de Restricción].
  2. El sistema mostrará los Datos Requeridos para el nuevo tipo de Restricción.
  3. Escribir la Descripción del Nuevo Tipo de Restricción. (Nombre de Tipo de Sanción)
  4. "Seleccionar" la Modalidad.
    • Economica: Se definira un monto fijo en Dinero.
    • Labor Social: Se definira un número de horas de Trabajo Social.
    • En Especie: Se compensara.
  5. Hacer "Clic" en la opción [Guardar Cambios].
  6. El sistema mostrará un mensaje de alerta.

<MODIFICACIÓN DE UN TIPO DE SANCIÓN>

  1. El Sistema muestra un filtro de Tipo de Sanciones, seleccionar el tipo de Filtro.
    • Mostrar Todos.
    • Económicas.
    • Labor Social.
    • En Especie.
  2. Hacer "Cilc" en el Nombre o Descripción de la Sanción.
  3. El sistema mostrará los datos requeridos para realizar la moificación.
  4. Hacer "Clic" en la opción [Guardar Cambios].
  5. El sistema mostrará un mensaje de alerta.

<BORRAR UN TIPO DE SANCIÓN >

  1. "Seleccionar" la casilla con el Nombre o descripción del Tipo de Sanción que se desea Borar.
  2. Hacer "Cilc" en la opción [Borar Tipo de Sanción].
  3. El sistema mostrará un mensaje de alerta.
