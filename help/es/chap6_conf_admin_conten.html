5. Configuración

5.2. Administración de Contenidos

En esta opción el Administrador del Sistema podrá administrar los contenidos de la pagina principal del sistema, los cuales son Notas, Ligas y Eventos.

Requisisto: El usuario administrativo deberá contar con el privilegio de Administración de Contenidos.

<AGREGAR NUEVA NOTA>

  1. Seleccionar la Pestaña [Notas].
  2. Hacer "Clic" en la opción de [Agregar Nueva Nota].
  3. Aparecerá un cuadro de dialogo donde se escribiran los datos de necesarios para agregar una nota nueva.
  4. Escribir la Etiqueta para mostrar.
  5. Seleccionar las Fechas en las que se mantendrá activa esa Nota.
  6. Escribir la informacioón breve de la Nota.
  7. Hacer "clic" en la opción [Guardar cambios].
  8. El sistema mostrará un mensaje de alerta, los datos fueron creados.

<AGREGAR NUEVO EVENTO>

  1. Seleccionar la Pestaña [Eventos].
  2. Hacer "Clic" en la opción de [Agregar Nuevo Evento].
  3. Aparecerá un cuadro de dialogo donde se escribiran los datos de necesarios para agregar un Nuevo Evento.
  4. Escribir la Etiqueta para mostrar.
  5. Seleccionar el Tipo de Evento.
  6. Seleccionar las Fechas en las que se mantendrá activo el Evento.
  7. Selecionar el lugar del evento.
  8. Seleccionar las Fechas de agenda del Evento.
  9. Escribir el horario.
  10. Escribir la breve información.
  11. Escribir la Información mas amplia.
  12. Hacer "clic" en la opción [Guardar cambios].
  13. El sistema mostrará un mensaje de alerta, los datos fueron creados.

<AGREGAR NUEVA LIGA>

  1. Seleccionar la Pestaña [Ligas].
  2. Hacer "Clic" en la opción de [Agregar una Liga].
  3. Aparecerá un cuadro de dialogo donde se escribiran los datos de necesarios para agregar un Nuevo Evento.
  4. Escribir la Etiqueta para mostrar.
  5. Seleccionar las Fechas en las que se mantendrá activa la Liga.
  6. Escribir el vínculo o dirección de la Página de Internet.
  7. Hacer "clic" en la opción [Guardar cambios].
  8. El sistema mostrará un mensaje de alerta, los datos fueron creados.
