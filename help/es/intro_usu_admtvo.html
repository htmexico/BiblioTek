Ingreso a BiblioTEK

Usuario Administrativo

El Usuario Administrativo es aquel que tendrá el control total del Sistema Bibliotek, incluyendo el Alta y Baja de otros usuarios asi mismo asignando los privilegios adecuados para cada usuario que tendrá control sobre el sistema.

REQUISITO:El Usuario Administrativo deberá contar con un Nombre de Usuario y Contraseña para tener acceso al Sistema BiblioTEK.

    1. Escribir el Nombre de Acceso al Sitema BiblioTEK.
    2. Escribir la Contraseña del Usuario.
    3. Hacer "Cilc" en la opción [Ingresar].
      • Si el Usuario y/o Contraseña es incorrecta, Mensaje de Alerta.
      • Si el Usuario y/o Contraseña es correcta, Acceso al Sistema BiblioTEK.

4. El sistema mostrará todas las opciones disponibes.
