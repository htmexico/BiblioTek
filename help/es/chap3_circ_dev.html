2.2. Devoluciones

En esta opción el Administrador del Sistema podrá realizar el proceso de Devolución de un Material en Préstamo a partir de un Nombre de Usuario.

REQUISITO: Para realizar una Devolución deberá contar con un Nombre de Usuario.

  1. Escribir el Nombre de Usuario, Teclee [Enter].
    • Si el usuario no existe el sistema mostrará un Mensaje de Alerta ("El usuario no fue encontrado").
    • Si el usuario existe, el sistema mostrará el Nombre Completo del Usuario. (Continuar Paso No. 2)
  2. Hacer "Clic" en la opción [Continuar].
  3. Aparecerá una pantalla mostrando los datos Requeridos para realizar la Devolución.
    • Si el usuario no presenta algún Material en Préstamo, el Sistema mostrará un Mensaje de Alerta.
    • Si el usuario presenta Material en Préstamo, el Sistema mostrará las características del Material. (Continuar Paso No.4)
  4. "Seleccionar" el Material que será Devuelto.
  5. Hcer "Clic" en la opción [Devolver los Ítems Señalados].
  6. El Sistema mostrará una pantalla que indicará que la Devolución fué Registrada y si desea un Nuevo Proceso de Devolución.

NOTA: El Botón con el Símbolo de Interrogación (?) , puede ayudarnos a Localizar o Consultar un Usuario de la Institución o un Ítem de un Material.
