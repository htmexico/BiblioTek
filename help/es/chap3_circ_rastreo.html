2.6. Rastreo

En esta opción el Administrador del Sistema podrá verificar por medio del ID del Título el Status de todas las copias del ID consultado.

REQUISITOS: Para realizar un Rastreo, es necesario contar con el ID del Título.

  1. Escribir el ID del Título, Teclee [Enter]. (NOTA: El ID del título es un Dato Númerico)
    • Si el ID no existe el sistema mostrará un Mensaje de Alerta.
    • Si el ID existe, (Continuar Paso: No. 2)
  2. Hacer "Cilc" en la opción [Continuar].
  3. El sistema mostrará un calendario de tres meses de disponibilidad con sus respectivos íconos. (NOTA: Los meses que se mostraran, Mes Actual, Mes Anterior y Mes Siguiente).

 

NOTA: El Botón con el Símbolo de Interrogación (?) , puede ayudarnos a Localizar o Consultar un Usuario de la Institución o un Ítem de un Material.
