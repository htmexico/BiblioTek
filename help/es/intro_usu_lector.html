Ingreso a BiblioTEK

Usuario Lector

El Usuario Lector es aquel que tendrá un control mínimo del Sistema Bibliotek, en el cual podrá realizar las Consultas tales como, Consulta Rápida del Material, Material Usado Recientemente, Temas más Frecuentados, Ultimas Contribuciones y Ultimas Actividades.

REQUISITO:El Usuario Lector deberá contar con un Nombre de Usuario y Contraseña para tener acceso al Sistema BiblioTEK.

  1. Escribir el Nombre de Acceso al Sitema BiblioTEK.
  2. Escribir la Contraseña del Usuario.
  3. Hacer "Cilc" en la opción [Ingresar].
    • Si el Usuario y/o Contraseña es incorrecta, Mensaje de Alerta.
    • Si el Usuario y/o Contraseña es correcta, Acceso al Sistema BibioTEK.

  4. El sistema mostrará todas las opciones disponibes.
