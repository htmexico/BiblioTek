3.5. Restricciones

En esta opción el Adminsitrador del Sistema podrá agregar o eliminar restricciones de usuario.

REQUISITOS: Para poder agregar o eliminar restricciones se deberá tener el nombre del usuario.

  1. Escribir el Nombre de Usuario, Teclee [Enter].
    • Si el usuario no existe el sistema mostrará un Mensaje de Alerta ("El usuario no fue encontrado").
    • Si el usuario existe, el sistema mostrará el Nombre Completo del Usuario. (Continuar Paso No. 2)
  2. Hacer "Clic" en la opción [Continuar].
  3. El sistema mostrará las restricciones del usuarrio.
  4. Si se desea agregar restricciones, hacer "clic" en la opción [Agregar restricción].
  5. Se mostrarán los datos requeridos para la restricción.
  6. "Seleccionar" la restricción que tendrá el usuario.
  7. "Seleccionar" las fechas desde y hasta, es el tiempo que tendra activa la restricción el usuario.
  8. Escribir la Descripción de la restrición del usuario.
  9. Hacer "Clic" en la opción [Guardar Cambios].
  10. Se mostrará un mensaje de alerta, Hacer "clic" en la opción [Aceptar].
  11. Se agregrá la restricción al Usuario y se notificará al usuario por vía email.

NOTA: El Botón con el Símbolo de Interrogación (?) , puede ayudarnos a Localizar o Consultar un Usuario de la Institución o un Ítem de un Material.
