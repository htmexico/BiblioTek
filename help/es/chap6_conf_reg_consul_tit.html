5. Configuración

5.9. Reglas para Consultas de Títulos

En esta opción el Administrador del Sistema podrá realizar la Configuración de las Consultas de Títulos, Agregar, Borrar y Modificar.

REQUISITO: El administrador del sistema deberá tener los suficientes privilegios para tener acceso a esta opción.

<AGREGAR NUEVA CONSULTA >

  1. Hacer "Clic" en el la opción [Nueva Consulta].
  2. El sistema mostrará los Datos Requeridos para el nuevo tipo de Restricción.
  3. Escribir el Nombre del nuevo Tipo de Restricción.
  4. "Selecionar" la Casilla de Activa.
  5. "Seleccionar" las Opciones que se desean incluir para la Consulta.
    • Palabras Clave.
    • Título.
    • Autor.
    • Materias.
    • Signatura Topográfica.
    • ISBN.
    • ISSN.
  6. "Seleccionar" cuantos registros se desea que se muestren por página de consulta.
  7. "Seleccionar" el Contenido Digital.
  8. "Seleccionar" el Tipo de Consulta.
    • Consulta de Acceso al Público.
    • Administtivos.
    • Usuarios de Consulta.
  9. "Seleccionar" si se permite:
    • Ver fichas de títulos.
    • Ver Copias existentes.
    • Ingresar a la información detallada de cada Copia.
  10. "Seleccionar" el estilo de Visualización.
    • Fichas MARC.
    • Fichas AACR2
    • Filtrados por Rango de Edad.
    • Filtrados por tipo de Material.
  11. "Seleccionar" la consulta será ordenada por:
    • Autor.
    • Título.
    • Signatura Topográfica.
    • Título y Edición.
  12. Hacer "Clic" en la opción [Guardar Cambios].
  13. El sistema mostrará un mensaje de alerta.

<MODIFICACIÓN DE UN TIPO DE SANCIÓN>

  1. Hacer "Cilc" en el Nombre o Descripción de la Consulta.
  2. El sistema mostrará los datos requeridos para realizar la modificación.
  3. Hacer "Clic" en la opción [Guardar Cambios].
  4. El sistema mostrará un mensaje de alerta.

<BORRAR UN TIPO DE SANCIÓN >

  1. "Seleccionar" la casilla con el Nombre o descripción de la Consulta que se desea Borrar.
  2. Hacer "Cilc" en la opción [Borrar Tipo de Sanción].
  3. El sistema mostrará un menasje de alerta.
